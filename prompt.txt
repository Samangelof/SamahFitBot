Требования к программе (обязательно соблюдать):

Тренировки:
Расписать план по дням с указанием: упражнения, количество подходов и повторений, отдых между подходами (в минутах), техника (если важно).

Обязательно добавить прогрессию нагрузки во 2-й неделе.

Питание:
Составить план по дням с разбивкой на приёмы пищи (завтрак, обед, ужин, перекусы).
Указать: количество граммов, КБЖУ каждого приёма, простые рецепты (учитывать лимит времени на готовку).
Дать варианты замены продуктов без потери КБЖУ.

Дополнительно:
Включить чек-листы для отслеживания прогресса (при желании пользователя).
Дать рекомендации по восстановлению: сон, вода, стретчинг.
Предложить конкретные способы мотивации, без воды.

Формат ответа:

Использовать маркированные списки.

Все единицы измерения: кг, см, минуты, граммы — строго соблюдать.

Важно:
Программа должна быть реалистичной, научно обоснованной.
Избегать клише вроде «проконсультируйтесь с врачом».
Давать чёткие, выполнимые инструкции без общих фраз.
